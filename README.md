# Zadanie
Stworzyć mały sklep internetowy (bez backendu) za pomocą React

## Wymagania
- Logowanie na sztywno jako user/user
- Jako zalogowany user możesz przeglądać ofertę
- Oferta z kilkoma przedmiotami, które sa gdzieś zamockowane
- Opcja filtrowania po nazwie, cenach i sortowanie rosnąco/malejąco
- Koszyk, gdzie wylądują wszystkie wybrane przedmioty przez użytkownika
